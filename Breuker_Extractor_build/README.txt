Breuker_Extractor is a Python script to extract Collision Unfolding Data acquired in timsTOF instruments. The software can only run headless (thru terminal) or can be run thru CIUSuite 3. Use tdfExtract for a standalone extraction tool for Breuker data with experimental parameters.
